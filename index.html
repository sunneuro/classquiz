Firebase Setup Guide

Step 1 — Create Firebase Project

1
Go to firebase.google.com and sign in with your Google account
2
Click Add project → name it classquiz → click through (disable Analytics is fine) → Create project
3
Once created, click the Web icon </> → give app nickname classquiz → click Register app
4
You will see a code block. Find and copy these 3 values:
apiKey, projectId, appId → paste into Setup → Firebase tab

Step 2 — Create Firestore Database

5
In Firebase left menu → Build → Firestore Database → Create database
6
Choose Start in test mode → Next → select any location → Enable
7
That's it — no further config needed. Test mode allows reading and writing for 30 days (enough to evaluate the app). You can extend it later.

Step 3 — Paste values & test

8
Go back to Setup → Firebase tab → paste the 3 values → click Test Firebase Connection → should show green ✅
9
Done! All student results will now flow into Firebase automatically. QR codes will be short and scannable.
